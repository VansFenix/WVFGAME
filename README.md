# WVF GAME

Игровой портал прямо в браузере. Змейка, тетрис, шахматы, судоку, казино и другие игры.

## Игры

- **Snake** — классическая змейка
- **Tetris** — тетрис с нарастающей скоростью
- **Chess** — шахматы с ИИ (minimax + альфа-бета отсечение)
- **Checkers** — шашки
- **Sudoku** — генератор с уникальным решением
- **TicTacToe** — крестики-нолики против ИИ
- **Memory** — игра на запоминание
- **Doom** — шутер от первого лица на canvas
- **Casino** — слоты, рулетка, кости, блэкджек

## Возможности

- Система валюты и ставок
- Профиль игрока, статистика, ежедневная награда
- PWA — установка на телефон как приложение
- Service Worker — работает офлайн
- Частицы на фоне, звуковые эффекты

## Технологии

- Vite
- Vanilla JS (ES-модули)
- Web Crypto API (SHA-256)
- Web Audio API
- Canvas 2D
- GitHub Pages

## Запуск

```bash
npm install
npm run dev     # разработка
npm run build   # сборка в dist/
```

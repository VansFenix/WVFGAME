# WVF GAME

Игровой портал прямо в браузере. 10 игр + казино с валютой 🪙.

## Игры

| Игра | Категория | Особенности |
|------|-----------|-------------|
| **Snake** 🐍 | Аркада | Классическая змейка, ускорение, D-pad |
| **Tetris** 🧊 | Классика | Hold, ghost, wall kicks, 7 фигур |
| **Flappy Bird** 🐦 | Аркада | Физика, трубы, рекорды |
| **2048** 🔢 | Логика | Canvas, свайп, плитки |
| **Chess** ♚ | Стратегия | ИИ minimax + alpha-beta, piece-square tables |
| **Checkers** 🟦 | Стратегия | ИИ negamax, обязательные взятия, дамки |
| **Sudoku** 🔢 | Логика | Генератор с уникальным решением, 3 подсказки |
| **TicTacToe** ❌ | Логика | ИИ minimax (непобедим), 2 игрока |
| **Memory** 🧠 | Память | 8 пар карт, таймер |
| **Doom** 🔥 | Экшн | Шутер, волны врагов, WASD + мышь |
| **Casino** 🎰 | Азарт | Слоты, кости, орёл/решка, блэкджек |

## Возможности

- 🪙 Система валюты — зарабатывай в играх, трать в казино
- 👤 Профиль игрока, статистика, уровни, ежедневная награда
- 🔇 Кнопка отключения звука (сохраняется)
- 📱 Полностью адаптивный дизайн (мобильные + десктоп)
- 📲 PWA — установка на телефон как приложение
- 📡 Service Worker — работает офлайн
- ✨ Частицы на фоне, aurora-эффекты, 3D-карточки
- 🔊 Звуковые эффекты через Web Audio API

## Технологии

- Vite 6
- Vanilla JS (ES-модули)
- Web Crypto API (SHA-256)
- Web Audio API (синтез звука)
- Canvas 2D
- GitHub Pages + GitHub Actions

## Запуск

```bash
npm install
npm run dev     # разработка (localhost)
npm run build   # сборка в dist/
```
